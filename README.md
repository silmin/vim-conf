please [read](.vimrc).

## usage
Please Rename or Remove `~/.vimrc` and `~/.vim` in advance.
```bash
$ cd ~
$ git clone https://github.com/silmin/vim-conf
$ ln -s vim-conf/.vimrc .vimrc
$ ln -s vim-conf/.vim .vim
```

next, open vim
```bash
$ vim
```
type `:PlugInstall`.
Plugin will be installed.

done.
